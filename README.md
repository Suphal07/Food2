# Food Calories AI Analyzer API

## Installation

1. Clone the repo:
   ```
   git clone https://github.com/Suphal07/Food2.git
   cd Food2
   ```
2. Install dependencies:
   ```
   npm install
   ```

## Usage

1. Create a `.env` file based on `.env.example`.
2. Start the server:
   ```
   npm start
   ```

## API Endpoints

- `POST /api/food/upload` - Upload food image.
- `GET /api/food/calculate` - Get calories for a food item.
